# LangChain - 50 вопросов для подготовки к собесу

Сборник вопросов по LangChain для подготовки к техническому собеседованию на позиции AI Engineer, ML Engineer, LLM Developer.

Покрывает основные темы, которые встречаются на собеседованиях по направлению LLM и AI-автоматизация.

## Темы

- **LCEL и цепочки** — Runnable, invoke / stream / batch, pipe `|`, retry, fallback
- **Prompts** — PromptTemplate, ChatPromptTemplate, partial, MessagesPlaceholder
- **Output Parsers** — PydanticOutputParser, StrOutputParser, JsonOutputParser, OutputFixingParser
- **Retriever и VectorStore** — Chroma, Faiss, Pinecone, MMR, MultiQueryRetriever
- **RAG** — pipeline, чанкинг, chunk_size, загрузка документов, fine-tuning vs RAG
- **Memory** — ConversationBufferMemory vs LangGraph checkpointer
- **Tools и Agents** — Tool, AgentExecutor, ReAct, LangChain Agent vs LangGraph
